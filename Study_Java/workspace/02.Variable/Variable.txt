1. 변수(Variable)
	- 데이터(값)를 저장할 때 사용되는 메모리의 기억공간
	- 데이터(값)를 담는 상자 : 크기 ▶ Data Type(자료형)
	- 기억공간에는 이름이 있어야 한다 ▶ 식별자
	
2. 변수 이름(식별자) 작성 규칙
	- 반드시 영문자 소문자로 시작
	- 첫글자는 영문자 소문자, '_', '$' 으로만 시작
	- 예약어(KeyWord)는 사용 불가
	- 띄어쓰기는 사용할 수 없다
	- 글자수에 제한이 없으므로, 가급적이면 이름에 의미가 나타나도록 작성하자
	- 카멜(Camel) 표기법 활용 : 단어가 연결될 경우 단어의 첫 글자를 대문자로 처리

★ 선언, 할당, 초기화, 나열, 재할당

3. 자료형(Data Type) : 변수의 크기와 저장되는 데이터의 종류를 결정
	① 기본 자료형(PDT) : 플랫폼에 상관없이 항상 고정된 크기를 갖는 자료구조
		- 정수형 : byte(1), short(2), int(4), long(8)
		- 실수형 : float(4), double(8)
		- 문자형(단일문자) : char(2)
		- 논리형 : boolean(1)		
		
	② 사용자 정의 자료형(UDDP) : 참조형 자료구조(Reference Type)
		- 클래스 타입
		- 나열(열거) 타입
		- 인터페이스 타입
